51 智能寻迹小车（标准描述）
基于51 单片机的智能寻迹小车，是一款以红外循迹为核心、可沿地面预设黑线自动行驶的小型轮式智能车，是嵌入式入门、电子实训、课程设计最经典的综合项目。
一、核心定义
以STC89C52/AT89S51为控制核心，通过红外反射传感器识别地面黑白轨迹，经单片机逻辑判断，驱动电机自动纠偏、沿黑线稳定行驶。
二、硬件组成（标准配置）
主控单元
51 单片机最小系统（晶振、复位、电源、下载口），负责信号采集、逻辑运算、电机控制。
路径检测
红外循迹模块（TCRT5000），常用2 路 / 3 路 / 4 路，利用黑白对红外反射率差异输出高低电平。
动力与驱动
直流减速电机 + L298N/L293D电机驱动板，实现前进、后退、左右转向、PWM 调速。
电源系统
7.4V 锂电池 / 4 节 AA 电池，为电机与控制系统供电。
机械结构
小车底盘、车轮、万向尾轮、传感器支架。
三、工作原理
红外发射管发出红外光；
白色地面强反射 → 接收管导通 → 输出低电平；
黑色轨迹弱反射 → 接收管截止 → 输出高电平；
单片机读取电平判断位置，控制左右电机差速纠偏，保持沿黑线行驶。
四、功能与控制逻辑
基础功能：自动寻迹、直行、转弯、过 S 弯、十字路口判断
典型逻辑：
中间传感器在线 → 直行
左侧传感器在线 → 右修正
右侧传感器在线 → 左修正
多路传感器可实现更平稳的PID 纠偏
五、软件特点
开发环境：Keil C51
语言：C 语言
核心模块：传感器读取、电机驱动、PWM 调速、循迹算法
可扩展：蓝牙遥控、避障、测速、OLED 显示
六、典型应用
单片机 / 嵌入式入门实训
电子设计竞赛基础车
自动导引小车（AGV）原理验证
课程设计 / 毕业设计
七、核心优势
成本低、资料极丰富、易上手
覆盖传感器、MCU、电机驱动、控制算法全流程
可快速扩展多功能智能车
